DronAnalyzer

======================================================================

Usage of tools:

1.Table overview - you can see all data in organized way

2.3D Visualization - you can see your whole flight in 3D environment

3.KML file - it creates file on your desktop, that can be ran with Google Earth

4.Graphs:

a.Alt/Time - change of altitude over time

b.Rol/Time - change of roll over time

c.Pitch/Time - change of pitch over time

d.Yaw/Time - change of yaw over time

e.Voltage and Alt/Time - change of voltage(in %) and altitude over time

Potential error when using DronAnalyzer:

●Make sure you have installed java on your system.

●If you are using windows, make sure your python3 installation is good. If it is, check if it is in your system path and make sure you have correctly downloaded numpy and open3D.

●If it can’t write files make sure you allowed it to do so in system settings.

●Make sure that your file path to log file does not have any spaces in names of folders (eg. name Drone Logs would cause and error but Drone_Logs won’t).

Thank you for using DronAnalyzer.
